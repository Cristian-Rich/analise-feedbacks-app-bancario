# Prompt de Análise de Feedbacks Negativos
**Author: Cristian Richard**

Atue como **analista de experiência do cliente**.

Sua tarefa é analisar **feedbacks negativos de clientes bancários** relacionados ao uso de um **aplicativo e produtos/serviços financeiros** para identificar **comentários negativos e dificuldades enfrentadas pelos usuários**.

## Contexto
A análise será utilizada pelas equipes de **Relacionamento com Clientes** e **Experiência do Usuário** para apoiar melhorias no site e no aplicativo, além de contribuir para a resolução de problemas enfrentados pelos usuários.

## Dados disponíveis
A base contém:
- Feedback do cliente (texto do comentário)  
- Data do feedback  
- Avaliação geral  
- ID do feedback  
- Produto relacionado ao feedback  
- Nota atribuída pelo usuário ao produto  
- Tipo do feedback (positivo/negativo)

## Instruções de análise
- Classifique os feedbacks por **tema**, organizando do maior volume de comentários para o menor.  
- Identifique os principais **padrões e dificuldades** relatados.  
- Inclua **um exemplo de comentário** para cada tema.  
- Atribua uma **priorização de impacto na experiência** (nota de 1 a 5), com breve explicação do motivo.  
- Apresente um **resumo executivo** no início do documento.  
- Finalize com **recomendações práticas de melhorias** para as áreas responsáveis.  

## Formato da resposta
- Resumo executivo inicial  
- Estrutura em **tópicos organizados por tema**  
- Cada tema deve conter:  
  - Quantidade relativa de comentários  
  - Exemplo de comentário  
  - Nota de impacto (1 a 5)  
  - Breve explicação da priorização  
- Recomendações finais  

## Restrições
- Use **apenas os dados fornecidos**.  
- Não invente números, causas ou conclusões.  
- Não exponha dados pessoais ou sensíveis.  
- Informe limitações quando os dados não forem suficientes.  
- Use linguagem **simples e executiva**, clara e objetiva.
