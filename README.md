# Análise de Feedbacks Negativos - Aplicativo Bancário
**Author: Cristian Richard**

Este repositório contém o **prompt estruturado** para análise de feedbacks negativos de clientes bancários relacionados ao uso de um aplicativo e produtos/serviços financeiros.

## Objetivo
Fornecer um guia claro para que a IA analise os comentários dos clientes, identifique dificuldades enfrentadas e organize os resultados em formato executivo, apoiando as equipes de Relacionamento e Experiência do Usuário.

## Conteúdo
- `prompt-analise-feedbacks.md`: Documento com o prompt final estruturado.  
- `README.md`: Explicação sobre o repositório e seu propósito.  

## Como usar
1. Copie o conteúdo do arquivo `prompt-analise-feedbacks.md`.  
2. Utilize-o em sua ferramenta de IA para realizar a análise dos feedbacks.  
3. Aplique os resultados para apoiar melhorias no aplicativo e nos serviços bancários.  

## Organização
Este repositório foi criado para servir como referência prática e organizada, permitindo versionamento e colaboração em análises de experiência do cliente.
